لتجاهل هذا التحذير، يمكنك ببساطة تجاهله عند استخدام التطبيق في البيئة التطويرية. ومع ذلك، إذا كنت تعتزم نشر التطبيق في بيئة إنتاج، فعليك اتباع إرشادات الاستضافة لـ Flask واستخدام خادم WSGI.

css
gunicorn -w 4 -b 0.0.0.0:5000 app:app

حيث:

  • app:app تحديد اسم التطبيق واسم متغير التطبيق في الملف "app.py".

باستخدام Gunicorn، يمكنك تجنب تحذير Flask واستخدام خادم WSGI مناسب للإنتاج.

from flask import Flask, request, jsonify
import pandas as pd
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity

app = Flask(__name__)

# بيانات الكورسات
courses_data = {
    'course_id': [1, 2, 3],
    'course_name': ['Python Programming', 'Machine Learning', 'Web Development'],
    'description': ['Learn Python programming basics', 'Introduction to machine learning algorithms', 'Build dynamic websites using HTML, CSS, and JavaScript']
}
courses_df = pd.DataFrame(courses_data)

# تحويل النص إلى متجهات Tfidf
tfidf_vectorizer = TfidfVectorizer()
tfidf_matrix = tfidf_vectorizer.fit_transform(courses_df['description'])

# مسار الصفحة الرئيسية
@app.route('/')
def home():
    return 'Welcome to the homepage!'

# مسار توصيات الكورسات
@app.route('/recommend', methods=['POST'])
def recommend_courses():
    user_input = request.form.get('query')
    if user_input:
        # تحويل نص الاستعلام إلى متجه Tfidf
        query_vector = tfidf_vectorizer.transform([user_input])

        # حساب التشابه الكوسيني بين نص الاستعلام ووصف الكورسات
        cosine_similarities = cosine_similarity(query_vector, tfidf_matrix)

        # الحصول على الكورسات الأكثر تشابهًا
        similar_courses = courses_df.iloc[cosine_similarities.argsort()[0]][::-1]

        # تحويل النتائج إلى قاموس
        recommended_courses = similar_courses.to_dict(orient='records')

        return jsonify({'courses': recommended_courses})
    else:
        return 'Please provide a query parameter.'

بالطبع، إليك كيفية تنفيذ طلب GET إلى API Flask في مشروع Angular باستخدام Angular HttpClient:

  1. في مكون Angular الذي تحتاج فيه إلى استدعاء API Flask، قم بإنشاء خدمة لإدارة الاتصال بالـ API. يمكنك تنفيذ هذه الخدمة باستخدام Angular HttpClient. قم بإنشاء ملف خدمة جديد باستخدام الأمر التالي:

    ng generate service api

  2. في ملف الخدمة الجديد (api.service.ts)، قم بكتابة الكود التالي لإرسال طلب GET إلى API Flask:

    typescript
    import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class ApiService {

  private apiUrl = 'http://localhost:5000/recommend';

  constructor(private http: HttpClient) { }

  getRecommendations(query: string): Observable<any> {
    return this.http.get<any>(`${this.apiUrl}?query=${query}`);
  }
}

  3. في المكون الذي تريد فيه استخدام الـ API (على سبيل المثال، مكون الصفحة الرئيسية)، قم بإدخال الخدمة التي أنشأتها (api.service.ts) واستخدمها لإرسال طلب GET إلى الـ API. على سبيل المثال، في ngOnInit() تابع المكون:

    typescript
    import { Component, OnInit } from '@angular/core';
import { ApiService } from './api.service';

@Component({
  selector: 'app-home',
  templateUrl: './home.component.html',
  styleUrls: ['./home.component.css']
})
export class HomeComponent implements OnInit {

  recommendations: any[] = [];

  constructor(private apiService: ApiService) { }

  ngOnInit(): void {
    this.getRecommendations();
  }

  getRecommendations(): void {
    const query = 'Python'; // يمكنك تغيير هذا الاستعلام بحسب احتياجاتك
    this.apiService.getRecommendations(query)
      .subscribe((data) => {
        this.recommendations = data.courses;
        console.log(this.recommendations); // تسجيل البيانات في وحدة التحكم للتحقق منها
      });
  }

}

هذا الكود يقوم بإرسال طلب GET إلى API Flask على المسار "/recommend" بمعلمة الاستعلام "query" وقيمة "Python". بعد استلام الاستجابة من الـ API، يتم عرض البيانات المسترجعة في وحدة التحكم (Console) للتحقق منها. يمكنك استبدال "Python" بقيمة الاستعلام التي تريدها.
